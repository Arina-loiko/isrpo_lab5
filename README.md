## Теги в HTML

Структура парного тега: `<тег>тело</тег>`

Пример:

```html
<h1>Это заголовок</h1>
```

## Базовые HTML-теги

Примеры:

```html
<h2>Заголовок</h2>
<p>Абзац текста</p>
<strong>Жирный</strong>
<em>Курсив</em>
<hr>
<a href="https://example.com">Ссылка</a>
<img src="example.jpg" alt="Описание">
```

## Списки

```html
<ol>
    <li>Первый пункт</li>
    <li>Второй пункт</li>
</ol>

<ul>
    <li>Элемент списка</li>
    <li>Ещё один элемент</li>
</ul>
```

## Таблицы

```html
<table border="1">
    <tr>
        <th>Заголовок</th>
    </tr>
    <tr>
        <td>Ячейка</td>
    </tr>
</table>
```

## Формы

```html
<form>
    <label for="name">Имя:</label>
    <input id="name" type="text" placeholder="Введите имя">
    <button type="submit">Отправить</button>
</form>
```

## Семантические теги

```html
<header>шапка сайта</header>
<nav>навигация</nav>
<main>основной контент</main>
<section>раздел</section>
<article>статья</article>
<footer>подвал сайта</footer>
```
